移动开发者服务平台

移动开发者服务平台的地址：http://mob.com/#/index

Ubuntu下

  1. 注册一个账号
  2. 点击网页的 “立即下载ShareSDk”
  3. 选中 “android的图标”, 在”头部的标题栏中点击sdk下载” 选中 “ShareSDK For Android “
  4. 点击 “SDK 下载” 选择要分享的软件
  5. 进入ShareSDK解压目录，打开“Share SDK for Android”目录，可以找到“QuickIntegrater.jar”，这个就是快速集成ShareSDK的工具

  6. 启动终端 cd进入“Share SDK for Android”目录，输入： java -jar QuickIntegrater.jar –>> 文档地址

  7. 进入ShareSDK解压目录，打开“Share SDK for Android”目录，可以找到“Sample 文件夹” 直接用Android studio 打开

    • 将需要的库拷贝带 lib文件夹
    • assets文件夹拷贝到 main文件夹
    • onekeyshare文件拷贝到 java 文件夹（与activity文件夹同级目录）
    • drawable文件夹拷贝到 res文件夹
    • 在配置文件中（AndroidManifest.xml）–>> 文档地址
    • assets文件 —>>> ShareSDK AppKey = “b21d3e800242” 我的头像进入后台 –> 创建应用 地址
